Interaction Descriptions

Hover On

Trigger

  • Hover on a button.

Rules

  • Cursor changes to pointer.
  • Over 300ms, the white button top will move down by 8px.

Feedback

  • Cursor changes to pointer.
  • Over 300ms, the white button top will move down by 8px.

Loops

  • None.

Modes

  • None.

Hover Off

Trigger

  • Hover off a button.

Rules

  • Cursor changes to default arrow.
  • Over 300ms, the white button top will move up by 8px.

Feedback

  • Cursor changes to default arrow.
  • Over 300ms, the white button top will move up by 8px.

Loops

  • None.

Modes

  • None.

Click: Springy Buttons

This variant feels right for buttons that are pressed and spring back, like a spacebar.
This will be used for buttons that open modals or turn pages.

Trigger

  • Click on a springy button.

Rules

  • Over 300ms, the white button top will move down by 8px, fully obscuring the gray button side.
  • Then, over another 300ms, the white button top will move up by 8px, returning to the hover position.

Feedback

  • Over 300ms, the white button top will move down by 8px, fully obscuring the gray button side.
  • Then, over another 300ms, the white button top will move up by 8px, returning to the hover position.

Loops

  • None.

Modes

  • None.

Click: Sticky Buttons

This variant feels right for buttons that are pressed and stick, like toggles in a switchboard.
This will be used for things like page selection in a navbar.

Trigger

  • Click on a sticky button.

Rules

  • Over 300ms, the white button top will move down by 8px, fully obscuring the gray button side.

Feedback

  • Over 300ms, the white button top will move down by 8px, fully obscuring the gray button side.

Loops

  • None.

Modes

    • None.
