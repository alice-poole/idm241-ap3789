IDM241 Alpha Description

Pinterest is one of my favorite websites. But its posts are a pain point for me every day.
First, let's take a look at the current microinteraction of hovering on a post here. (Posts are the main content blocks of a page.)


Current Design

Hover On

Trigger

  • Hover on a post.

Rules

  • Cursor changes to magnifying glass.
  • Post darkens slightly.
  • Save button appears.
  • Predicted list dropdown appears.
  • Original post link button appears.
  • Share button appears.
  • More options button appears.

Feedback

  • Cursor changes to magnifying glass.
  • Post darkens slightly.
  • Save button appears.
  • Predicted list dropdown appears.
  • Original post link button appears.
  • Share button appears.
  • More options button appears.

Loops


Modes

  • Save button appears.
  • Predicted list dropdown appears.
  • Original post link button appears.
  • Share button appears.
  • More options button appears.

Hover Off

Trigger

  • Hover off a post.

Rules

  • Cursor reverts to arrow.
  • Post lighting resets.
  • Save button disappears.
  • Predicted list dropdown disappears.
  • Original post link button disappears.
  • Share button disappears.
  • More options button disappears.

Feedback

  • Cursor reverts to arrow.
  • Post lighting resets.
  • Save button disappears.
  • Predicted list dropdown disappears.
  • Original post link button disappears.
  • Share button disappears.
  • More options button disappears.

Loops


Modes

  • Save button disappears.
  • Predicted list dropdown disappears.
  • Original post link button disappears.
  • Share button disappears.
  • More options button disappears.

Problems

  1. Tint obscures image content - the focus of the website!
  2. Instant transition is boring for a site about art & expression.

Proposed Design

Hover On

Trigger

  • Hover on a post.

Rules

  • Cursor changes to magnifying glass.
  • Save button slides down from top by 16px then back 4px.
  • Original post link slides up from bottom by 16px then back 4px.
  • Share button slides up from bottom by 16px then back 4px.
  • More options slides up from bottom by 16px then back 4px.

Feedback

  • Cursor changes to magnifying glass.
  • Save button slides down from top by 16px then back 4px.
  • Original post link slides up from bottom by 16px then back 4px.
  • Share button slides up from bottom by 16px then back 4px.
  • More options slides up from bottom by 16px then back 4px.

Loops


Modes

  • Save button appears.
  • Original post link button appears.
  • Share button appears.
  • More options button appears.

Hover Off

Trigger

  • Hover off a post.

Rules

  • Cursor reverts to arrow.
  • Save button slides up out of view.
  • Original post link slides down out of view.
  • Share button slides down out of view.
  • More options button slides down out of view.

Feedback

  • Cursor reverts to arrow.
  • Save button slides up out of view.
  • Original post link slides down out of view.
  • Share button slides down out of view.
  • More options button slides down out of view.

Loops


Modes

  • Save button disappears.
  • Original post link button disappears.
  • Share button disappears.
  • More options button disappears.

Advantages

  1. Image content stays bright and clear with less clutter.
  2. Unified style and animation of UI elements offer tighter design language.
  3. Restricting text to buttons improves visual contrast for accessible text.
  4. Lively feedback draws attention to post saving and sharing functionalities.
