IDM241 Final Description

Hover On

Trigger

  • Hover on a button.

Rules

  • Cursor changes to pointer.
  • Over 200ms, the white button top will move down by .25rem.

Feedback

  • Cursor changes to pointer.
  • Over 200ms, the white button top will move down by .25rem.

Loops

  • None.

Modes

  • None.

Hover Off

Trigger

  • Hover off a button.

Rules

  • Cursor changes to default arrow.
  • Over 200ms, the white button top will move up by .25rem.

Feedback

  • Cursor changes to default arrow.
  • Over 200ms, the white button top will move up by .25rem.

Loops

  • None.

Modes

  • None.

Default Select On Page Load

Trigger

  • Load the page.

Rules

  • Select the first button with the "default" tag. See "Option Selected" below. This brings additional rules and feedback.

Feedback

  • Over 200ms, the white button top will move down by .25rem, fully obscuring the gray button side.

Loops

  • None.

Modes

  • None.

Click Springy Next Button

Trigger

  • Click on the springy next button.

Rules

  • Over 200ms, the white button top will move down by .25rem, fully obscuring the gray button side.
  • Select the option button to the immediate right of the current selected button. See "Option Selected" below. This brings additional rules and feedback.
    • (This wraps around. If the rightmost option button is already selected, instead, select the leftmost option.)
  • Then, over another 200ms, the white button top will move up by .25rem, returning to the hover position.

Feedback

  • Over 200ms, the white button top will move down by .25rem, fully obscuring the gray button side.
  • Then, over another 200ms, the white button top will move up by .25rem, returning to the hover position.

Loops

  • None.

Modes

  • None.

Click Sticky Option Button

Trigger

  • Click on a sticky option button.

Rules

  • Over 200ms, the white button top will move down by .25rem, fully obscuring the gray button side.
  • Select the clicked button. See "Option Selected" below. This brings additional rules and feedback.

Feedback

  • Over 200ms, the white button top will move down by .25rem, fully obscuring the gray button side.

Loops

  • None.

Modes

  • None.

Option Selected

Trigger

  • An option is selected, which can occur through the following means:
    • A default option is selected on page load.
    • The next button is clicked, toggling the selection one to the right.
    • An option button is clicked directly.

Rules

  • Over 200ms, the screen background color will fade transition to a hidden color attached to the newly selected option button.
  • Over 200ms, excluding the newly selected button, all sticky buttons' white button tops will move to .5rem above their gray button side. (Any previously selected sticky button resets to the default unpressed position.)

Feedback

  • Over 200ms, the screen background color will fade transition to a hidden color attached to the newly selected option button.
  • Over 200ms, excluding the newly selected button, all sticky buttons' white button tops will move to .5rem above their gray button side. (Any previously selected sticky button resets to the default unpressed position.)

Loops

  • None.

Modes

  • None.
